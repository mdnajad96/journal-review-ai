# 🗄️ Database Setup\n\n## SQL Schema\n\nRun this in Supabase SQL Editor:\n\n```sql\n-- Create users table\nCREATE TABLE users (\n  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),\n  email VARCHAR(255) UNIQUE NOT NULL,\n  full_name VARCHAR(255),\n  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP\n);\n\n-- Create papers table\nCREATE TABLE papers (\n  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),\n  user_id UUID NOT NULL REFERENCES users(id),\n  title VARCHAR(512) NOT NULL,\n  authors TEXT[] DEFAULT ARRAY[]::TEXT[],\n  abstract TEXT,\n  source VARCHAR(50) NOT NULL,\n  file_url TEXT,\n  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP\n);\n\n-- Create analyses table\nCREATE TABLE analyses (\n  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),\n  paper_id UUID NOT NULL REFERENCES papers(id),\n  user_id UUID NOT NULL REFERENCES users(id),\n  analysis_type VARCHAR(50) NOT NULL,\n  content TEXT NOT NULL,\n  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP\n);\n\n-- Create indexes\nCREATE INDEX idx_papers_user_id ON papers(user_id);\nCREATE INDEX idx_analyses_paper_id ON analyses(paper_id);\nCREATE INDEX idx_analyses_user_id ON analyses(user_id);\n```\n\n## How to Apply\n\n1. Go to Supabase Dashboard\n2. Click \"SQL Editor\"\n3. Click \"New Query\"\n4. Paste the SQL above\n5. Click \"Run\"\n\n## Row Level Security (RLS)\n\n```sql\n-- Enable RLS\nALTER TABLE papers ENABLE ROW LEVEL SECURITY;\nALTER TABLE analyses ENABLE ROW LEVEL SECURITY;\n\n-- Papers policy\nCREATE POLICY papers_select ON papers\n  FOR SELECT USING (user_id = auth.uid());\n\nCREATE POLICY papers_insert ON papers\n  FOR INSERT WITH CHECK (user_id = auth.uid());\n\n-- Analyses policy\nCREATE POLICY analyses_select ON analyses\n  FOR SELECT USING (user_id = auth.uid());\n\nCREATE POLICY analyses_insert ON analyses\n  FOR INSERT WITH CHECK (user_id = auth.uid());\n```\n\n## Database Tables\n\n### users\n- id (UUID)\n- email (VARCHAR)\n- full_name (VARCHAR)\n- created_at (TIMESTAMP)\n\n### papers\n- id (UUID)\n- user_id (FK)\n- title (VARCHAR)\n- authors (TEXT[])\n- abstract (TEXT)\n- source (VARCHAR)\n- file_url (TEXT)\n- created_at (TIMESTAMP)\n\n### analyses\n- id (UUID)\n- paper_id (FK)\n- user_id (FK)\n- analysis_type (VARCHAR)\n- content (TEXT)\n- created_at (TIMESTAMP)\n\n---\n\n**Database ready!** 🗄️\n
