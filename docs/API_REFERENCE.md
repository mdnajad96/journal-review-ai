# 📚 API Reference\n\n## Base URL\n```\nhttp://localhost:3000/api\nhttps://your-deployed-app.com/api\n```\n\n## Upload Paper\n\n**POST** `/api/upload`\n\nUpload a PDF or Endnote file\n\n**Request:**\n```javascript\nconst formData = new FormData();\nformData.append('file', file);\nformData.append('type', 'pdf'); // or 'endnote'\n\nconst response = await fetch('/api/upload', {\n  method: 'POST',\n  body: formData,\n});\n```\n\n**Response:**\n```json\n{\n  \"success\": true,\n  \"file\": {\n    \"name\": \"paper.pdf\",\n    \"size\": 2048576,\n    \"type\": \"application/pdf\"\n  }\n}\n```\n\n## Analyze Paper\n\n**POST** `/api/analyze`\n\nAnalyze text with AI\n\n**Request:**\n```javascript\nconst response = await fetch('/api/analyze', {\n  method: 'POST',\n  headers: { 'Content-Type': 'application/json' },\n  body: JSON.stringify({\n    text: 'Journal text here...',\n    analysisType: 'summarize' // or keyFindings, polishProposal\n  })\n});\n```\n\n**Response:**\n```json\n{\n  \"success\": true,\n  \"result\": \"AI analysis result...\"\n}\n```\n\n## Search Papers\n\n**GET** `/api/search`\n\nSearch for papers\n\n**Query Parameters:**\n```\n?topic=machine+learning   (required)\n&source=arxiv             (optional: arxiv, crossref, all)\n&limit=10                 (optional: max results)\n```\n\n**Example:**\n```javascript\nconst response = await fetch(\n  '/api/search?topic=quantum+computing&source=arxiv&limit=5'\n);\n```\n\n**Response:**\n```json\n{\n  \"success\": true,\n  \"papers\": [\n    {\n      \"title\": \"Paper Title\",\n      \"authors\": [\"Author 1\", \"Author 2\"],\n      \"publishedDate\": \"2024-01-15\",\n      \"abstract\": \"Abstract text...\",\n      \"url\": \"https://arxiv.org/...\",\n      \"source\": \"arxiv\"\n    }\n  ]\n}\n```\n\n## Error Responses\n\n```json\n{\n  \"error\": \"Error message here\"\n}\n```\n\n## Status Codes\n\n- 200 OK - Success\n- 400 Bad Request - Missing required parameter\n- 500 Server Error - Server error\n\n---\n\n**Ready to build!** 🚀\n
