Using in Code\n\n```typescript\nimport { summarizeJournal } from '@/lib/ai-service';\n\nconst result = await summarizeJournal(journalText);\nif (result.success) {\n  console.log(result.data); // AI summary\n}\n```\n\n## Available Functions\n\n### summarizeJournal(text: string)\nGenerates a 2-3 paragraph summary\n\n```typescript\nconst summary = await summarizeJournal(paperText);\n// Returns: \"This paper explores...\"\n```\n\n### extractKeyFindings(text: string)\nExtracts 5 most important findings\n\n```typescript\nconst findings = await extractKeyFindings(paperText);\n// Returns: \"1.
