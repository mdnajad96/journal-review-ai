# 🚀 Deployment Guide\n\n## Option 1: Vercel (Recommended - 5 minutes)\n\n### Why Vercel?\n- Free tier perfect for students\n- 1-click deployment from GitHub\n- Automatic HTTPS\n- Auto-deploys on GitHub push\n- GitHub Student Pack benefits\n\n### Steps:\n\n1. **Sign up at Vercel** (free)\n   - Go to https://vercel.com\n   - Click \"Sign up\" → \"GitHub\"\n   - Grant permissions\n\n2. **Import Your Repository**\n   - Click \"New Project\"\n   - Select `journal-review-ai`\n   - Click \"Import\"\n\n3. **Configure Environment Variables**\n   - Add:\n     ```\n     OPENAI_API_KEY=sk-your-key\n     NEXT_PUBLIC_SUPABASE_URL=https://...\n     NEXT_PUBLIC_SUPABASE_ANON_KEY=...\n     ```\n   - Click \"Deploy\"\n\n4. **Done!** 🎉\n   - Live at `https://journal-review-ai.vercel.app`\n   - Auto-deploys on every GitHub push\n\n---\n\n## Option 2: Railway\n\n### Steps:\n\n1. Go to https://railway.app\n2. Click \"New Project\"\n3. Select \"Deploy from GitHub repo\"\n4. Choose `journal-review-ai`\n5. Add environment variables\n6. Deploy!\n\n---\n\n## Option 3: Netlify\n\n### Steps:\n\n1. Go to https://netlify.com\n2. Click \"New site from Git\"\n3. Select `journal-review-ai`\n4. Build settings:\n   - Build: `npm run build`\n   - Publish: `.next`\n5. Deploy!\n\n---\n\n## Environment Variables\n\n```env\nOPENAI_API_KEY=sk-your-key-here\nNEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co\nNEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key\n```\n\n---\n\n## Custom Domain\n\n1. Buy domain (Namecheap, GoDaddy, etc.)\n2. Point DNS to Vercel\n3. Vercel Dashboard → Settings → Domains\n4. Add custom domain\n5. Wait 24-48 hours\n\n---\n\n## After Deployment\n\n- [ ] Test all features\n- [ ] Check error logs\n- [ ] Verify API keys work\n- [ ] Test on mobile\n- [ ] Share with friends!\n\n---\n\n**Your app is live!** 🌍\n
